# Helpdesk — Front-end

Angular 16 front-end for a helpdesk/support ticket system.

> Back-end repo: [helpdesk-back](https://github.com/andremendoncaolv/helpdesk-back)

---

## Requirements

- Node.js 18+
- Angular CLI 16

```bash
npm install -g @angular/cli@16
```

## How to Run

```bash
git clone https://github.com/andremendoncaolv/helpdesk-front.git
cd helpdesk-front
npm install
ng serve
```

Navigate to `http://localhost:4200/`. The app reloads automatically on file changes.

## Build

```bash
ng build
```

Output is generated in the `dist/` directory.

## Running Tests

```bash
ng test        # Unit tests via Karma
ng e2e         # End-to-end tests
```
